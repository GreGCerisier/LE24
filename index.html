La mise en ligne sous gitlab

Step 1 : configurer sa présentation

  1. Mettre la présentation dans un dossier public
  2. A la racine du projet rajouter un fichier .gitlab-ci.yml

.gitlab-ci.yml


pages:
  stage: deploy
  tags:
  - shell
  script:
  - echo 'Deploy to Gitlab pages'
  artifacts:
    paths:
    - public
  only:
  - master

Step 2 : Se faire un repos sous gitlab

Step 3 : Commit

Step 4 : Done

C'est pas plus compliqué que ça :)

La mise en ligne sous github

  1. Faire un repos sout github
  2. Commit la présentation sur une branche gh-pages
  3. Done
  4. Presentation accessible depuis : yourname.github.io/repo-name
